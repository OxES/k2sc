# K2SC

Python package for the K2 systematics correction using Gaussian processes.

## Installation

    git clone https://github.com/OxES/k2sc.git
    cd k2sc
    python setup.py install --user

## Requires

 - NumPy
 - SciPy
 - PyFITS
 - George
